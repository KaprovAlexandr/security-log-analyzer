# Security Log Analyzer

## Цель

Автоматизация анализа Linux auth.log для обнаружения подозрительной активности

## Возможности

- поиск brute-force
- поиск sudo
- успешный вход после failed
- статистика ip
- экспорт отчета

## Стек / Инструменты

- python
- regex
- linux auth.log

## Как запустить

python main.py auth.log
